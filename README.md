
#  Wanderlust

Wanderlust is a travel listing web app built to explore and share destinations around the world. Users can browse places, create their own listings, upload images, and leave reviews — making it a simple platform for documenting travel experiences.

This project focuses on building a complete CRUD application with authentication, image uploads, and clean UI/UX.

🔗 Live Project: https://wanderlust-1-yj19.onrender.com/listings

---

## ✨ What you can do

* Browse travel destinations
* Create your own listings
* Upload images for places
* Edit or delete your listings
* Add reviews and ratings
* Secure login and signup
* Only owners can modify their listings
* Responsive UI for different screen sizes
